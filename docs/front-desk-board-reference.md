# Front-desk board reference

What action each Needs-action state needs.

- **Held** — clock is on the room. Convert before it lapses: confirm if the guest is ready, or let it release.
- **Reply overdue** — guest is waiting on you. Answer now: Confirm if available, Cancel if not. Never leave it silent.
- **Awaiting (not yet overdue)** — safe, still in the response window. Confirm when you get to it, don't sit on it.
- **Arriving / due** — guest is showing up. Highest priority. Make sure they're confirmed and check-in ready.
- **Fresh / new** — just came in, no rush. Confirm in normal order.
